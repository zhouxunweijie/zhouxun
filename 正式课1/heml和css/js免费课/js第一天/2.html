什么是js? 定义：js是javascript缩写，他是一种基于对象和事件的脚本语言（开发语言） js是运行在客户端的一种开发语言（js没有所谓的安全性） 开发语言：是指可以实现一定的功能，js是一门（轻量级）的开发语言 js是做什麽的？ 呈现动态效果。数据交互 js的组成？ （1）ECMAscript js的API里面规定了js最基本的知识，数据类型。命名规范，操作语句等 （2）DOM document object model 文档对象模型--》DOM是操作document(文档对象) document(文档对象)包括了所有这个文档下面的元素（html下面所有的标签） （3）BOM browser object model 浏览器对象-->BOM是操作 window(浏览器对象) js的命名规范？ 1.严格区分大小写 2.使用驼峰命名法 （1）两个以上有意义的单词组合的时候，第一个单词的首字母小写，其余有意义的单词首字母大写 2）可以使用数字，_,$,字母来命名，不能使用数字做首位 3）不能使用关键字和保留字 关键字：在js中有特殊意义的 保留字：未来有可能成为关键字的 html如何引入JS文件？ 1.行内式 将JS代码块写在标签内 onclick鼠标点击事件 2.内嵌式 将js代码写在script标签中间的 //js的注释 //js的多行注释 3.外链式 通过script标签中的scr这个标签属性引入一个js的外部文件 内嵌式和外链式都要写在body的结束标签之前（整个文档最后） 代码块写在外链式的标签中是无效的 引入js代码尽量使用外链式（一次封装多次调用） js的输出？ 1.alert（‘内容’); 在浏览器的弹出窗口弹出我们的内容 2.console.log(‘内容’); 在浏览器的控制台中输出内容 3.console.dir(‘内容')； 在浏览器的控制台中输出内容（还输出对象的属性和方法--将这个对象详细的打印出来） 4.console.table('内容')；在浏览器的控制台以表格的形式输出内容 5.document.write（“”）；直接在文档中输出内容 6.innerHTML 将内容动态添加到指定元素内--可以识别标签 7.innerText 将内容动态添加到指定元素内--只能识别文本 js中的变量？ 用来代表值和储存值的 变量可以理解为一个盒子，可以向这个盒子放任何东西，盒子本身没有意义，里面装什么他就是什么 格式：声明 变量名=值； var num=1;num就代表1 里面储存的也是1 为什么变量可以放任何东西-->以为js中的变量是松散数据类型的 js中的数据类型 基本数据类型：数字(number) 字符串(string) 布尔(boolean) null(空 以后会有) undefined（以后也没有） 引用数据类型： 1.对象：数组类（Array） 对象类(Object) 正则类(RegEXP) 字符串类(String) 时间类(Date) 2.函数：function 数字：number 负数 小数 0 NaN 字符串：string 用引号包起来的内容就是字符串 字符串本身没有任何意义“”/‘’ 布尔：boolean 只有两个值 true（真）/false（假） null:null undefined:undefined 数组：array[]包起来的，里面的每一项都用逗号隔开[function(){},1,obj={}] 对象:object{}包起来的 里面是成对的属性值和属性名 里面用逗号隔开 正则：regexp /^$/用来匹配的 函数：就是一个方法 定义+调用 定义：function 函数 数字数据类型：number 包括 正数 负数 小数 0 NaN {not a number 不是一个有效数字} 判断一个值是否为数字-->isNaN(要判断的值) 返回结果：1）true{不是一个有效数字} 2）false{是一个有效数字} isNaN()在判断之前，会强制转化-->Number();
