ps:函数值定义不执行的时候，没有任何意义 函数可以多次被执行，并且每次执行的时候和上一次没有任何关系 function fn(){ console.log(1); } fn(); fn(); 函数的运行机制: 1、当代码在浏览器中加载的时候，浏览器会提供一个全局作用域供这些代码从上到下执行 2、定义函数部分--浏览器提供一个新的内存空间，将函数体中的代码当做字符串存到这个空间中，将这个空间地址返回给这个函数，现在函数内存的不是代码，而是一个空间地址 3、函数执行部分（将函数中的代码执行），这个时候我们需要去之前的内存空间进行调取，浏览器会提供一个新的空间让调取的代码从上到下执行 ps：函数可以一次定义多次执行，那么每一次执行的时候，都会去最开始的那个空间去取代码，那么这个空间就会一只被占用，我们将这个一直被占用的空间叫做堆内存 函数运行时浏览器提供的空间，运行后机会被销毁，我们将这种用完即销毁的空间叫做栈内存 函数的参数： 函数获取外部值的时候，就会用到函数的参数 function 函数名(形参){} 函数名(实参);
